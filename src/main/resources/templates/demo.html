GBM Challenge - Taxi tracking application

Dummy client used to send requests to the web service.

This application has 3 registered users. You can use one to test all the functionalities.

There are three functionalities

  • Test user - Test the credentials and recovery the user's last location.
  • Show historical - Show all user's locations, grouped by travel.
  • Start tracking - Start a new tracking. This functionality sends several requests to the web service:
    • new-tracking - (one request) Initialize a new tracking.
    • set-position - (multiple requests) Send a new location every several seconds.
    • In addition, this functionality communicates with other interested parties using a webhook each time that the user's location is updated.

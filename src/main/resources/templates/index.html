GBM Challenge - Taxi tracking

Final application

  • Taxi tracking application

Technical details

  • /get-last-position - (GET request) Get the last position registered of the user who made the request. Return JSON.
  • /get-historical - (GET request) Get all trackings and registered locations of the user who made the request. Return JSON.
  • /new-tracking - (GET request) Start a new tracking. Return the new travel's ID.
  • /set-position - (PUT request) Set a new position to tracking.
  • It uses a basic access authentication (user & password).
  • Website to visualize the webhook requests
